The Weighbridge API

⚡

Real-Time Weight Capture

Stream live weight readings from any indicator at <50ms latency via WebSocket or HTTP polling endpoints.

🔗

Multi-Protocol Support

Native drivers for MODBUS RTU/TCP, RS232, RS485, USB-HID, and Ethernet indicators from all major brands.

📋

Transaction Management

Create, update, and query weighbridge dockets with full vehicle, product, and tare management built in.

🔐

Secure & Authenticated

JWT-based auth with role-level permissions, audit logs, and end-to-end encryption for all transactions.

weighbridge_api.js
// Initialize WeighBridgePro API client const WeighBridge = require('weighbridgepro-sdk'); const client = new WeighBridge({ apiKey: process.env.WBP_API_KEY, endpoint: 'https://api.weighbridgepro.io' }); // Capture a live weight reading async function captureReading(scaleId) { const reading = await client.scales.getReading({ id: scaleId, units: 'tonnes', stable: true }); return { gross: reading.gross, // 48.920 t tare: reading.tare, // 14.200 t nett: reading.nett, // 34.720 t timestamp: reading.ts }; } // Create a docket / transaction await client.transactions.create({ vehicle: 'ABC-1234', product: 'Iron Ore', supplier: 'Acme Mining Co', reading: reading });
